
One step install - Spark & Iceberg

Features

1. Devcontainer for easy setup: locally and one codespaces
2. Jupyter notebooks via VScode
3. Pyspark - Run and debug
4. Data engineering extensions: e.g. SQLtools and data wrangler to view parquet files

Prerequisites

1. git
2. docker and docker compose
3. VSCode 
4. >= 4GB RAM

Uses

https://hub.docker.com/r/tabulario/spark-iceberg
